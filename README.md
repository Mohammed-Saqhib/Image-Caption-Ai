## ‚ú® Key Features

### AI Captioning & OCR
- **Smart Captioning**: Uses the Salesforce BLIP model to understand and describe images in natural language.
- **Text Extraction**: Integrated EasyOCR to detect and extract text from images, supporting multiple languages.

### Accessibility & Localization
- **Text-to-Speech (TTS)**: Converts generated captions into audio with support for multiple accents and languages.
- **Real-time Translation**: Translates captions into Hindi, Kannada, Spanish, French, German, and Italian.

### Performance Optimization
- **Cloud API Mode**: Offloads heavy processing to Hugging Face Inference API for faster results on low-resource devices.
- **Local Mode**: Runs models locally for privacy and offline capability (requires sufficient hardware).
- **Optimized Audio**: Smart caching and random ID generation for seamless audio playback.

## üö¶ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mohammed-Saqhib/Image-Caption-Ai.git
   cd Image-Caption-Ai
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**
   ```bash
   streamlit run streamlit_app.py
   ```

4. **Open in browser**
   - The app will automatically open at `http://localhost:8501`
