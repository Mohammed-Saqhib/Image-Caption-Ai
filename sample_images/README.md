# Sample Images

This folder contains sample images for testing the Image Caption AI application.

## Usage

- Place your sample images here (PNG, JPG, JPEG, GIF, WEBP)
- Images will be available in both Streamlit and Web frontend
- Users can select from samples or upload their own

## Recommended Samples

For best demonstration:
1. **Text Images**: Images with readable text (signs, posters, documents)
2. **Landscape Photos**: Scenic views, nature shots
3. **People Photos**: Portraits, group photos
4. **Product Images**: Items, objects with clear details
5. **Mixed Content**: Images with both text and visual elements

## Image Guidelines

- **Format**: PNG, JPG, JPEG, GIF, WEBP
- **Size**: Under 10MB recommended
- **Resolution**: Any resolution (will be processed automatically)
- **Quality**: Higher quality = better results

## Quick Start

1. Add your sample images to this folder
2. Restart the application
3. Select "Use Sample Image" option
4. Choose from available samples
5. Generate caption!

---

**Note:** Add at least 3-5 diverse sample images for best user experience.
